# Tic-Tac-Toe Hub

A collection of Tic-Tac-Toe games featuring different difficulty levels, including an unbeatable AI, playable via web interface.

## Features
- **Easy Bot:** Random move generation.
- **Mid Bot:** Rule-based logic with basic strategy.
- **Hard Bot:** Unbeatable AI using Minimax Algorithm.
- **Web Interface:** Built with PyWebIO for an interactive browser experience.

## How to Run Locally
1. Clone this repository.
2. Install dependencies:
   ```bash
pip install pywebio


Run the main file:

Bash
python main.py
Deployment
This project is configured to run on Render. Ensure you have the requirements.txt file included in your repository.
